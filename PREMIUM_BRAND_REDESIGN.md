## ✨ Typography

### **Font Families:**
- **Headings:** Playfair Display (serif, elegant, expressive)
- **Body:** Inter (clean, modern, readable)
- **Buttons:** Inter Medium with letter-spacing

### **Font Sizes:**
- H1: 3.5rem (4rem in hero)
- H2: 2.5rem (3rem in CTA)
- H3: 1.75rem
- Body: 1rem with 1.75 line-height
- Hero description: 1.35rem

### **Font Weights:**
- Headings: 600-700 (expressive)
- Body: 400 (regular)
- Hero/CTA text: 300 (light, elegant)
- Buttons: 500 (medium)

## 🎨 UI Styling Details

### **Spacing:**
✅ Lots of white space (breathing room)  
✅ Clean layouts with generous padding  
✅ Sections: 6rem top/bottom padding  
✅ Cards: 3rem padding  

### **Rounded Corners:**
```css
--radius-sm: 8px    /* Buttons, small elements */
--radius-md: 12px   /* Cards, service items */
--radius-lg: 16px   /* Large components */
```

### **Shadows (Soft & Subtle):**
```css
--shadow-sm: 0 2px 4px rgba(0,0,0,0.3)
--shadow-md: 0 4px 8px rgba(0,0,0,0.4)
--shadow-lg: 0 8px 16px rgba(0,0,0,0.5)
--shadow-xl: 0 12px 24px rgba(0,0,0,0.6)
```

### **Hover Animations:**
✅ Smooth cubic-bezier transitions (0.4s)  
✅ Slight scale on cards (translateY -8px)  
✅ Brightness increase on buttons  
✅ Border color shifts to wine red
