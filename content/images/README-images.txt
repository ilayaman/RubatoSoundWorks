# Images Folder

## How to Add Images

Place your studio photos and project images in this folder. Then reference them in your content files.

### Studio Photos

Add your studio photos with names like:
- `studio-1.jpg` - Main recording room
- `studio-2.jpg` - Control room
- `studio-3.jpg` - Vocal booth
- `studio-4.jpg` - Equipment

### Project Images

Add project-related images:
- `project-1.jpg`
- `project-2.jpg`
- etc.

### Usage in Markdown Files

Reference images in your markdown files like this:

```markdown
![Alt text](/images/studio-1.jpg)
```

Or with a caption:

```markdown
![Studio Control Room](/images/studio-2.jpg)
*Our state-of-the-art control room*
```

### Recommended Image Sizes

- Studio photos: 1920x1080px (16:9 ratio)
- Project thumbnails: 800x600px
- Logo/Icons: 512x512px

### Supported Formats

- JPG/JPEG (recommended for photos)
- PNG (recommended for graphics with transparency)
- WebP (for smaller file sizes)
- GIF (for simple animations)

### Optimization Tips

Before uploading images:
1. Resize to appropriate dimensions
2. Compress using tools like TinyPNG or ImageOptim
3. Use descriptive file names (e.g., `recording-session-2024.jpg` instead of `IMG_1234.jpg`)
